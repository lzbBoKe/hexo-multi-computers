今天一个朋友给我发了句：
```
“青山一道同风雨，明月何曾是两乡。”
```
我很纳闷，我就上网查了一下，发现这是小日本援助武汉时印在物资上的。
<!--more-->
我没去过日本，搞不懂小日本是个什么国家。不过我估摸着，他们也就国家层面稍微懂一点中国诗歌，普通知识分子也不会对中国诗词有感情。

物资上还有：
```
“山川异域，风雨同天。”
```
```
“岂曰无衣，与子同裳。”
```
有机会，想去日本看看，调研一下当地的生活！
